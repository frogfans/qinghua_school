学习心得分享

经典的课程体系和雄厚的师资队伍、经验丰富的教务团队、创新的教学方式，注重理论联系实践的教学理念，帮助企业家同学不仅是学习知识，更重要是解决问题。
2013年，我回到深圳开始担任康佳广东冰洗市场负责人，明显感觉到自己遇到了提升瓶颈。于是，我决心再学习，经过几番比较，我选择了深圳清华大学研究院高级工商管理研修班项目。经过一年多的学习，深感再学习之必要，主要收获有：1、聆听名师讲课帮助我对时代变化、企业变革、经营管理有更深理解，温故知新、厚积薄发，这有助于建立更高水平的思考思维习惯；2、通过同学们的交流、思想碰撞，进一步打开了视野。现在我作为一名清华学生、非常荣幸，也为自己选择清华而庆幸！
——龙志勇 康佳集团营销管理部 总经理
仿佛时空穿越，我进入研究院学习，从一名商人成了一名学生，被清华百年的历史底蕴所深深浸泡和熏陶。班里的同学都是社会之中坚、企业之精英。通过学习与交流，不但开阔了我的管理视野，更改变了我的人生态度与生活方式。清华的丰富内涵及名师的亲点，让我慢慢地从创业初的激情走向理性，从果敢走向从容，更明了“自强不息，厚德载物”的企业家精神与使命。清华人，这将是我今后最自豪的称谓。
——黄舜 邦德教育集团董事、深圳博尔思文化传播有限公司董事长
深圳清华大学研究院是一个大平台，在这个平台上，一方面，我们常问自己有什么可以帮助到同学，另一方面，遇上困难时也主动请求同学帮助，群策群力地解决问题。更重要的是，在这个平台上，同学之间可以包容地指出彼此的局限，然后相互提升、共同修炼。这可能就是清华平台的价值，不同于公司，也不同于家庭，同学在这个平台上平等地成长、成熟、成功。大家都是百战归来再读书的人，童心、真心、爱心，这“三心”最重要。祝愿清华同学真诚在一起，分享学习，快乐成长。
——李良耀 东江集团董事
难忘每次老师精彩的授课，难忘每次同学热烈的讨论。在清华，我们收获了系统而丰富的知识，收获了老师的严谨与专业，收获了难以割舍的友谊，收获了搏击未来的勇气和力量。在清华一年半的学习，使我们的思想和心灵得到了升华，让我们面对未来激烈的市场竞争，多了一份从容和淡定。
——黄瑞云 深圳市艺星珠宝有限公司 副总经理
一年多的工商管理学习，得到了许多清华名师、专家的专业指导，他们理论水平扎实，有的老师本身就是企业家，担任上市公司或大型企业高级顾问，所以他们贴近企业实际的讲解，让同学产生很多共鸣，课堂互动氛围浓厚，我们企业面临的困惑和瓶颈得到了很好的指引，未来发展思路也更明晰了。
——邹二伟 深圳市华夏龙物流有限公司董事总经理
