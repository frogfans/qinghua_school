关于力合商学院

       力合科创集团，作为深圳清华大学研究院的主平台企业，负责实施研究院整体发展战略，形成了科技服务、投资并购、科技金融、科技产业、创新基地、国际合作、教育服务等业务板块齐头并进、深度融合的良好发展格局。
       力合商学院承担着高素质创新人才培养的重要职能，将抓住清华大学新百年发展战略以及研究院未来20年建设国际一流研究院的战略机遇，利用十几年丰富的人才培养经验，搭建更具有国际化、专业化和体系化的产学研资一体化高端教育平台。

项目背景

经国务院学位办[2004]67号文批准，同意中国云南财经大学与英国格林威治大学( University   of  Greenwich )合作培养项目管理专业和房地产专业两个理学硕士学位教育项目。2007年12月两项目通过中华人民共和国教育部复核，取得《中华人民共和国中外合作办学项目批准书》（编号：MOE53GB1A20040070O）。(中华人民共和国教育部教育涉外监管信息网：http://www.jsj.edu.cn )

合作院校

格林威治大学创立于1890年，是一所历史悠久、学风淳朴的国际化综合性大学，被评价为全英国最优秀的两所新兴大学之一,前身为英国皇家海军学院。现有20000名来自80多个国家的学生，她拥有英国最好的建筑与园林设计专业，她的校园建筑群号称英国的凡尔赛。

云南财经大学是中国政府认可的具有经济学、管理学、哲学和法学的学士、硕士学位授予权的大学，是云南省重点大学之一。本合作项目具体承办单位云南财经大学国际工商学院是专门进行国际教学交流与合作的教学单位。与英国格林威治大学合作培养硕士研究生项目两个，具有较强的国际合作办学经验和条件。详细资料请见网址：HTTP://WWW.YNUFE.EDU.CN

成绩考核及学位授予

学员完成规定课程学习，成绩合格；并按要求完成学位论文，可申请授予英国格林威治大学项目管理理学硕
士学位或房地产理学硕士学位。同时可办理中国教育部留学服务中心学历学位认证。
在申请硕士学位过程中，不需参加国家组织的研究生统一考试，不需参加论文答辩。


  • 入学条件

    具有国民教育系列的大学本科学历，或专科毕业须满三年的在职人员。

  • 入学流程

    报名——>提交材料——>录取——>缴费——>开学——>上课

  • 教学安排

    学制两年；主要学习地点：深圳、昆明

  • 毕业文凭

    学员完成学业，可申请授予英国格林威治大学项目管理理学硕士学位或房地产理学硕士学位。同时可办理中国教育部留学服务中心学历学位认证。
