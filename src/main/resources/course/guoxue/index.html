力合商学院

        力合商学院（力合教育）成立于2004年，是力合科创集团（深圳清华大学研究院主平台企业）八大业务板块中不可分割的部分，在力合科创的科技企业综合服务体系中承担着高素质创新人才培养的重要职能，力合商学院（力合教育）将抓住清华大学新百年发展战略以及研究院未来20年建设国际一流研究院的战略机遇，利用十几年丰富的人才培养经验，搭建更具有国际化、与业化和体系化的产学研资一体化高端教育平台。

课程背景：无处不在的“国学”

如果 你认为 国学在现代社会离你很遥远，
那么你错了
如果 你认为 国学只有在拥有丰富阅历、达到一定年龄、职级或境界后才需触及，
那么你又错了
国学，其实无处不在！

  • 国学是你快速了解政策导向的需要


    2015年3月国内首部聚焦习近平总书记用典的新书《习近平用典》在京首发，其中收录了习主席在各种讲话场合治国理政理念典故135则。不懂国学，有可能听不懂主席讲话；

  • 国学是你和孩子之间的一架桥梁


    国学中蕴含了历史名师圣人的智慧精华，是亲子教育、培养孩子“礼义廉耻孝悌忠信”的根据；国学教育已纳入青少年必修课，不懂国学，很可能缺失了辅导孩子国学功课的知识和能力；

  • 国学是现代企业经营管理的迫切需要


    随着企业的发展、在学习了西方管理理念和管理工具之后，越来越多的企业家发现，企业在发展到一定阶段后，企业管理其实是中学为体、西学为用。而无论企业的规模和发展阶段如何，企业发展的重要动力之一在于创始人的人格魅力和文化精神，而国学将在这些方面祝你和你的企业一臂之力；

国学经典学堂汇聚国学各领域的专家学者，旨在帮助更具雄心壮志和远见的经营管理者等优秀人士综览古今思想智慧，解读历史变化得失，修炼自己的思维，提升自己的境界，厚实思想积累，丰盈精神家园，寻回真我，成就卓越人生。

课程设置

围绕国学的七大经典设计课程，如下图所示。本期课程在原基础上新增了锦上添花的课题

  • A01儒学：内圣外王

    主要内容：

    1、经典精读：《四书》；                                                   2、儒学与人文素养；                                                         3、传统文化圣地体悟                                                           4、儒家智慧与现代生活                                                           5、儒家的内圣外王之道

    重要收获：

    儒学是中华传统文化的主脉，它陶冶、教化了中国人两千多年。课程着眼于儒家的内圣外王之道，阐述企业家如何丰富和发展自己的内心与人生，达到格物、致知、诚意之境，能修身有成、齐家有余，进而有能力兼济天下，超越自我，领受心灵的欢乐与平和。

    学时：16

    授课地点：山东曲阜孔子故里；深圳

  • A02心学：修心明觉

    主要内容：

     1、传奇王阳明；  2、心外无物； 3、良知；                          4、 本然与明觉； 5、致良知；    6、知行合一

    重要收获：

    中国历史上立德、立功、立言最为杰出的有两个半人!王阳明和孔子各占一席（曾国潘算半个），王阳明的心学是儒学中的一个门派。心学对为人、从商从业、治国理政、人生及世界观有着重要的启发。毛泽东思想的形成，心学也起到了重要的作用；蒋介石和蒋经国也是用心学的道理治理台湾；心学传到日本、韩国等东亚国家，对学术界影响很大。2015年习主席谈到“王阳明的心学正是中国传统文化中的精华”。

    学时：16

    授课地点：深圳

  • A03道学：道法自然

    主要内容：

    1、经典精读：《道德经》；2、逍遥道学与养生智慧；3、传统文化圣地体悟；4、道家的生存智慧；5、道家无为智慧与管理应用

    重要收获：

    道家思想崇尚自然，有辩证法的思维，同时主张清静无为，反对斗争。道家思想曾一度影响治国兴邦和人们为人处事，所蕴含的自然、无为的管理和立身养心的智慧在教育、经济等各方面都被借鉴和应用。

    学时：16

    授课地点：四川青城、峨眉，深圳

  • A04儒道易融合

    主要内容：

    1、经典精读《人物志》；           2、才性分类；                 3、人物鉴别；                            4、创业人才“英雄”论

    重要收获：

    《人物志》是唯一流传至今的中国古代人才学理论著作。该书综合儒道易学知识，论述了各类人才在生理素质、气质性格和管理才能方面的差异，以及适宜从事的职业和所能担当的职务；详细介绍了如何从形貌神态、言谈举止、为人处事等方面鉴别人才，并对鉴别中容易出现的失误及其原因进行了深入探索。为现代企业识人、用人提供了历史借鉴

    学时：16

    授课地点：深圳

  • A05兵学：制胜谋略

    主要内容：

     1、经典精读：《孙子兵法》；                                            2、中国的兵家智慧；                                                        3、战略思维与科学决策；   

    重要收获：

    中国的军事思想历史受中国哲学的影响，而中国军事思想中的辩论法又丰富与推进着中国哲学的发展。兵法不是小花招，而是谋略，是大战略、大智慧，因此，兵法自然成为了企业界不可或缺的商战参谋。

    学时：16

    授课地点：深圳

  • A06史学：知古鉴今

    主要内容：

    1、经典精读《资治通鉴》，给领导者的借鉴                            2、经典精读《史记•货殖列传》，商道智慧

    重要收获：

    “以史为鉴，可以知兴替”，从《资治通鉴》看王朝诸侯立业、兴业到衰败，以古鉴今，可助现代企业管理者究天人之际，察古今之变，明存亡之道，晓兴衰之理...商人史书《史记•货殖列传》是中国历史上第一部专门为商人树碑立传的史书，深刻揭示了商业经营、企业管理的本质和规律，被历代企业家奉为经商之圣经。南怀瑾曾说，“诸位老板，读书首先要读《货殖列传》！《货殖列传》里面，每一个故事都告诉你管理的道理。他中间提了很多怎么样做生意发财，怎么致富的”。

    学时：16

    授课地点：深圳

  • A07医学：健康养生

    主要内容：

     1、经典精读：《黄帝内经》；                                            2、黄帝内经与养生智慧；                                                    3、中医智慧与现代中医养生

    重要收获：

    《黄帝内经》是中国影响最大的一部医学著作，被称为中医之始祖。中医是中华民族传统文化的瑰宝，她不仅止于治病，更强调关注人的体格，保持健康、预防疾病，同时还进一步发扬和传承了中华文化精髓，强调阴阳五行、天人合一，从精、气、神合一的角度去认识活生生的人。

    学时：16

    授课地点：深圳

  • A08佛学：转识成智

    主要内容：

    1、经典精读：《金刚经》；                                               2、禅智慧与生命关怀；                                                       3、传统文化圣地体悟                                                              4、禅与智慧人生；                                                             5、禅文化与生命关怀

    重要收获：

    佛教传入中国以后，与中国文化由冲突到融合，最终达到宗教与哲学的高度统一。佛学中依正不二、佛性平等、菩萨心行等种种智慧和精神将帮助我们释放苦闷压抑的心灵，领悟到生命的真谛，圆满幸福的人生

    学时：16

    授课地点：浙江普陀山，深圳

  • A09增值模块：西方借鉴

    主要内容：

    1、西方文化纵览；                                                                 2、西方哲学与人类文明；                                                   3、西方哲学文化与现代商业

    重要收获：

    西方当代商业文明与其文化及哲学观密不可分，作为人类文化的重要组成部分，可以与中国传统文化相互借鉴，并为现代商业和企业管理者所用。

    学时：16

    授课地点：深圳

学习感言

国学是中国人与生俱来的一种遗传基因，是中华祖先千百年来注入我们身体当中的愿力与灵魂。我们应当激活国学基因，解码前辈的愿与投射其中的精神。我最初之所以选择国学是因为清华国学研究院最早成立于1926年，在新文化运动的大时代背景下，坚持用科学方法研究与传承国学。国学经典学堂1+7的课程体系与架构设计，国学智慧与国学应用统领全局，给学生宏观视野，易-儒-医-道-兵-佛-史，如同北斗七星。授课老师具有严谨的治学态度与高深的学术水平，不是作为培训，而是当作教育，老师备课都非常精心。


通过学习国学，可以帮助我们一明道，即总结规律，预测未来，消除内心的不确定，多一份安宁与平静。当今的时代是一个移动互联网、大数据、云计算、物联网的时代。从五千年文明的大数据中去总结规律，以史为鉴。二解惑。我们今天的烦恼与问题，前有都曾有相似的境况与思考，如何识别自己的情绪、调适自己的情绪、超越自己的情绪？通过学习国学经典，让我从中找到答案。三励志与增强民族自信，并不一定通读所有国学经典，往往是国学中的一句话激活了人的内在基因，联通了人的内心，给人内心坚定与强大就值得了。老子的一句话给了我很大的力量：“知人者智，自知者明；胜人者力，胜己者强。”无论何人，总是在入世追逐，要做到不断超越而不孤独，唯有不断超越自己，内心才会不断强大、自由。通过对国学的学习可以让人感受到中国传统文化、思想、精神的凝练与博大，五行中增强了民族自信。在学习的过程中，同学之间的交流也让人受益匪浅，俗话说三人行必有我师。


----陈登坤  金蝶集团执行董事、高级副总裁、金蝶医疗总经理


在参加国学经典学堂之前，对国学有兴趣但总感觉迷迷糊糊，进入深圳清华大学研究院学国学之后，跟着老师对7大经典国学经典进行精读，包括儒学的《四书》，道学的《道德经》，医学的《黄帝内经》等7大经典，在学习过程中我理解了儒家的最高境界， 感悟了道家老子提出的顺应规律的为人处世的方法：“上善若水”、“少则得，多则惑”、“夫唯不争，故天下莫能与之争”……“无为而治”， “治大国如烹小鲜”等对企业管理者授权和领导力的启示 ，也进一步了解了保持身心健康的日常预防、诊断等方法……这一切将惠及我的事业、生活的各个角落，将是我一生的财富，终生受益。


----贺  磊  深圳市大乘科技股份有限公司 董事长                    


对于一个人的成长来说，其所树立的价值观是至关重要的，因为一个人的价值观会决定他一生的走向。中国的传统文化主张“天人合一，道法自然，和为贵”等，这些理念在本质上都是与宇宙，人生的发展大道相一致的。因此，树立以中国传统文化为核心的价值观，可以确保我们的一生都能沿着健康，正确的方向上发展。近两年的国学经典班的学习，在我生命成长的过程中，可谓是一座里程碑，它让我从根本上掌握了生命的主动权！ 感恩中华文化瑰宝，我将不遗余力发扬！再次感谢“国学经典学堂”。


----沈  伟 深圳市体育中心 业务经营部部长                            


面对这个不断变化的世界，源于对中国传统文化的热爱，2012年的9月，国学经典学堂，我们来了。在中华5000年浩瀚瑰丽的文化宝藏面前，在学院安排的一大批著名高校和研究机构的资深学者、教授的讲授下，我们从经典解读开始，如饥似渴、吸收国学的营养和精华。易学、儒学、道学、史学、医学、佛学等七大模块，从不同角度和视野解读这个生活的世界以及生活的本质和精髓。国学的学习，不仅在课堂，课后我们志同道合的同学在学院的倡导下，积极组织进一步的研讨、交流，我们要对国学的学习一直继续下去，谢谢我们的引路人——国学经典学堂，是你激发了我们的国学热情，我们将终身受益。


----罗贤强  深圳市路尚投资管理有限公司执行董事 总裁
